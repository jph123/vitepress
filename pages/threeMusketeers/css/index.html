CSS

层叠样式表（Cascading Style Sheets），让网页变得美观动人。以下是 CSS 面试高频知识点。

高频面试题

01说说 CSS 盒模型▼

CSS 盒模型由 content（内容）、padding（内边距）、border（边框）、margin（外边距） 组成。

  • 标准盒模型（W3C）：box-sizing: content-box，width 只包含 content
  • IE 盒模型：box-sizing: border-box，width 包含 content + padding + border
/* 推荐全局使用 border-box */
* { box-sizing: border-box; }
面试要点：border-box 让宽度计算更直观，实际开发中几乎都会全局设置 border-box。
02什么是 BFC？如何创建 BFC？▼

BFC（Block Formatting Context，块级格式化上下文）是一个独立的渲染区域，内部元素的布局不会影响外部。

BFC 的特性：

  • 内部 Box 垂直方向上一个接一个放置
  • BFC 区域不会与浮动元素重叠
  • 计算 BFC 高度时，浮动元素也参与计算
  • BFC 内外 margin 不会折叠

创建 BFC 的方式：

/* 常用方式 */
overflow: hidden | auto | scroll;
display: flow-root;     /* 专为创建 BFC 设计 */
position: absolute | fixed;
float: left | right;
display: flex | grid | inline-block;
经典应用：清除浮动、防止 margin 塌陷、实现自适应两栏布局。
03CSS 选择器优先级是怎样的？▼

优先级从高到低：

  1. !important — 最高优先级（慎用）
  2. 内联样式 — style="..."（1000）
  3. ID 选择器 — #id（100）
  4. 类 / 伪类 / 属性选择器 — .class、:hover、[type]（10）
  5. 标签 / 伪元素选择器 — div、::before（1）
  6. 通配符 — *（0）
相同优先级时，后写的覆盖先写的（层叠规则）。
04如何实现元素水平垂直居中？▼

方案一：Flex（最常用）

.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}

方案二：Grid

.parent {
  display: grid;
  place-items: center;
}

方案三：定位 + transform

.child {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%);
}

方案四：定位 + margin auto（需固定宽高）

.child {
  position: absolute;
  top: 0; right: 0; bottom: 0; left: 0;
  margin: auto;
  width: 200px; height: 200px;
}
05说说 Flex 布局的常用属性▼

容器属性（设在父元素上）：

  • flex-direction：主轴方向 row | column | row-reverse | column-reverse
  • justify-content：主轴对齐 flex-start | center | space-between | space-around | space-evenly
  • align-items：交叉轴对齐 stretch | center | flex-start | flex-end | baseline
  • flex-wrap：是否换行 nowrap | wrap | wrap-reverse
  • gap：间距 gap: 16px

项目属性（设在子元素上）：

  • flex-grow：放大比例（默认 0，不放大）
  • flex-shrink：缩小比例（默认 1，可缩小）
  • flex-basis：初始大小（默认 auto）
  • flex: 1 = flex: 1 1 0%（等分剩余空间）
  • align-self：单独设置交叉轴对齐
06position 的值有哪些？区别是什么？▼
  • static：默认值，正常文档流，top/left 无效
  • relative：相对定位，相对自身原位置偏移，不脱离文档流
  • absolute：绝对定位，相对最近的非 static 祖先定位，脱离文档流
  • fixed：固定定位，相对浏览器视口，脱离文档流
  • sticky：粘性定位，滚动到阈值前是 relative，到达后变 fixed
/* sticky 经典用法：吸顶导航 */
.header {
  position: sticky;
  top: 0;
  z-index: 100;
}
07如何清除浮动？▼

方案一：BFC 清除浮动

.clearfix { overflow: hidden; }
/* 或 */
.clearfix { display: flow-root; }

方案二：伪元素清除浮动（最推荐）

.clearfix::after {
  content: '';
  display: block;
  clear: both;
}

方案三：额外标签法

<div style="clear:both"></div>
推荐使用伪元素方案，不影响布局且无副作用。
08如何实现响应式设计？▼
  • 媒体查询：@media (max-width: 768px) { ... }
  • 弹性单位：使用 rem、em、vw、vh、% 代替 px
  • Flex / Grid 布局：自适应弹性布局
  • 响应式图片：img { max-width: 100%; height: auto; }
  • 移动优先：先写移动端样式，再用 min-width 逐步增强
/* 移动优先示例 */
.grid { grid-template-columns: 1fr; }
@media (min-width: 768px) {
  .grid { grid-template-columns: 1fr 1fr; }
}
@media (min-width: 1024px) {
  .grid { grid-template-columns: 1fr 1fr 1fr; }
}
09重绘和回流（重排）的区别？如何避免？▼
  • 回流（Reflow / Layout）：元素的大小、位置发生变化，触发重新布局。开销大
  • 重绘（Repaint）：外观变化（颜色、背景等），不影响布局。开销较小

回流一定会触发重绘，重绘不一定触发回流。

避免方法：

  • 使用 transform 代替 top/left 动画
  • 使用 visibility: hidden 代替 display: none
  • 批量修改 DOM：使用 DocumentFragment 或先 display:none
  • 避免逐条修改样式，使用 CSS class 切换
  • 避免频繁读取触发回流的属性（offsetTop、scrollTop 等）
10CSS3 有哪些新特性？▼
  • 选择器：:nth-child()、::before、:not() 等
  • 圆角与阴影：border-radius、box-shadow
  • 渐变：linear-gradient()、radial-gradient()
  • 变换：transform: translate/rotate/scale/skew
  • 过渡：transition
  • 动画：@keyframes + animation
  • 弹性布局：flex
  • 网格布局：grid
  • 自定义属性：--var + var()
  • 媒体查询：@media
/* CSS 变量 */
:root {
  --primary: #1572b6;
}
.btn {
  background: var(--primary);
  transition: all 0.3s ease;
}
.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(21,114,182,.3);
}
