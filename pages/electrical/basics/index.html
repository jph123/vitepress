电气基础

直流电与交流电、交流接触器、中间继电器等电气基础知识点整理。

知识点

⚡直流电（DC）与交流电（AC）▼

直流电 DC（Direct Current）：大小、方向都不随时间变化。

交流电 AC（Alternating Current）：大小、方向随时间周期性变化。

记忆要点：DC = 不变；AC = 周期性变化。
🔌交流电分类：单相电与三相电▼

单相电：一根火线 + 一根零线

  • L — 火线（Live）
  • N — 零线（Neutral）

三相电：三根火线

  • L1、L2、L3 — 三根火线
单相电： L ────┐ N ────┘ 负载 三相电： L1 ───┐ L2 ───┤ 负载 L3 ───┘
家庭用电一般为单相电（220V），工业用电多为三相电（380V）。
🔧交流接触器 KM▼

工作原理：利用电磁力与弹簧弹力相配合，实现触头的接通和分断。

两种工作状态：

  • 失电状态（释放状态）：线圈无电，触头处于初始位置
  • 得电状态（动作状态）：线圈通电，电磁力吸合触头

三个组成部分：

  • 线圈：通电产生电磁力
  • 主触点：接通/分断主电路（大电流）
  • 辅助触点：用于控制电路（小电流），包括常开（NO）和常闭（NC）
交流接触器结构： ┌─────────────────────┐ │ 线圈 │ │ (A1 ---- A2) │ ├─────────────────────┤ │ 主触点 │ 辅助触点 │ │ L1-L2-L3│ NO NC │ └─────────────────────┘
总结：线圈得电 → 常开吸合，常闭断开；线圈失电 → 所有常开常闭复位。

复位：即常开恢复为常开，常闭恢复为常闭。
🛠️中间继电器 KA▼

三个组成部分：

  • 线圈：通电产生电磁力
  • 常开触点（NO）：线圈得电时闭合
  • 常闭触点（NC）：线圈得电时断开

工作逻辑：

  • 线圈得电 → 常开吸合，常闭断开
  • 线圈失电 → 常开闭合复位（恢复断开），常闭复位（恢复闭合）
中间继电器 vs 交流接触器： 中间继电器 KA 交流接触器 KM ───────────── ───────────── 触点容量小 触点容量大 用于控制电路 用于主电路 放大信号/增加触点 直接控制负载
记忆口诀：得电——常开变闭合，常闭变断开；失电——全部回原位。
📋触点状态速查表▼

无论是交流接触器 KM 还是中间继电器 KA，触点逻辑相同：

线圈状态 常开触点（NO） 常闭触点（NC）
得电 吸合（闭合） 断开
失电 复位（断开） 复位（闭合）
常开（NO = Normally Open）：常态下断开，得电后闭合。
常闭（NC = Normally Closed）：常态下闭合，得电后断开。
