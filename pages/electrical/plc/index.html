PLC 基础知识

常开常闭、置位复位、上升沿下降沿、定时器、数据类型等 PLC 核心知识点整理。

知识点

⚡常开（NO）和常闭（NC）▼

常开（NO = Normally Open）

  • 原始状态（按钮没按、线圈没电）：断开，线路不通
  • 动作后（按钮按下、线圈得电）：闭合，线路导通
俗称：一动就合，平时断开。

常闭（NC = Normally Closed）

  • 原始状态（按钮没按、线圈没电）：闭合，线路通
  • 动作后（按钮按下、线圈得电）：断开，线路切断
俗称：一动就断，平时导通。
常开 NO 常闭 NC 原始状态： 原始状态： ──/ ── (断开) ──| ── (闭合) 动作后： 动作后： ──| ── (闭合) ──/ ── (断开)
🔒置位（S）/ 复位（R）▼

置位（S = Set）：自锁，一直有电。即使触发信号消失，线圈仍然保持得电状态。

复位（R = Reset）：清零/断电，让它没电。解除自锁状态。

西门子置位 S、复位 R 线圈下方的数字 N，代表连续操作多少个位，取值范围：0~255。
置位 S：按下 → 线圈得电并保持（自锁） 松手 → 线圈仍然有电 复位 R：按下 → 线圈断电 松手 → 线圈仍然没电 典型应用：电机启停控制 启动按钮 → S（置位）→ 电机运行 停止按钮 → R（复位）→ 电机停止
📈上升沿与下降沿▼

上升沿（EU / PLS / ↑）

触点由断开变接通的那一瞬间，信号从 OFF→ON（0→1），瞬间产生一个扫描周期的脉冲信号，维持 PLC 1 个扫描周期，之后立刻断开。

下降沿（ED / PLF / ↓）

触点由接通变断开的那一瞬间，信号从 ON→OFF（1→0），瞬间产生一个扫描周期的脉冲信号，维持 PLC 1 个扫描周期，之后立刻断开。

信号波形： ┌──────┐ ┌────── │ │ │ ──────┘ └──────────────┘ ↑ ↑ ↑ 上升沿 下降沿 上升沿 (0→1) (1→0) (0→1) 脉冲输出： │ │ │ ───┘ └──────────────┘ 1个 1个 1个 扫描周期 扫描周期 扫描周期
核心要点：上升沿/下降沿只在一个扫描周期内有效，常用于触发一次性动作，如计数、状态切换。
⏱️定时器（TON / TOF / TONR）▼

接通延时定时器（TON）

按下按钮，到达预定的时间后电机启动；松手电机立刻停。

TON 接通延时： 按钮 ────────────────────┐ 松手 │ 定时 ────|====设定时间====|── 输出 ┌──┐ ───────────────────────┘ └── 立刻断开

断电延时定时器（TOF）

按下按钮，计时器常开触点立刻吸合通电；松开按钮后，到达预定时间常开才断开。

TOF 断电延时： 按钮按下 ──────┐ 松手 │ 输出 ──────────┤============设定时间====┐ │ 延时后才断开

带保持的接通延时定时器（TONR）

按下按钮会记录当前按下的时间，下次按下时会从上次记录的时间继续累计计时。需要手动复位才能清零。

TONR 带保持延时： 按下 3s ── 松开 ── 再按下 2s ── 松开 ── 再按下 │ 累计计时 = 3s + 2s + ... 到达设定时间后输出

📊PLC 数据类型▼
数据类型 长度（bit） 范围
位（Bool）10 / 1
字节（Byte）80 ~ 255
字（Word）160 ~ 65535
双字（DWord）320 ~ 4294967295
整数（Int）16-32768 ~ 32767
双整数（DInt）32±2147483647

记忆口诀：Bool 1 位、Byte 8 位、Word/Int 16 位、DWord/DInt/Real 32 位。Word 无符号、Int 有符号。
数据类型层级关系： 位 Bool (1 bit) └─ 字节 Byte (8 bit = 8 个 Bool) └─ 字 Word (16 bit = 2 个 Byte) └─ 双字 DWord (32 bit = 2 个 Word) 有符号版本：Int (16 bit) / DInt (32 bit) 小数版本： Real (32 bit 浮点数)
